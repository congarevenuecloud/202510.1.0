Description

The Recipients component allows users to manage document signers and recipients signature workflows.

Users can add multiple recipients, specify their email addresses, assign signer types, and optionally set a signing order for sequential document processing.

import { RecipientsModule } from '@congarevenuecloud/elements';

@NgModule({
  imports: [RecipientsModule, ...]
