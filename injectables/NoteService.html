Description

Note service provides functionality for managing Comments/Notes across business objects.

Usage

import { NoteService, AObjectService} from '@congarevenuecloud/ecommerce';

export class MyComponent implements OnInit{
constructor( private noteService: NoteService)
}
// or
export class MyService extends AObjectService {
private noteService: NoteService = this.injector.get(NoteService);
}

addComment
addComment(note: Note)

Adds a new comment/note to a specific record

Example:

const newNote = new Note();
newNote.Name = 'Comment Title';
newNote.ObjectId = 'proposal123';
newNote.ObjectType = 'Proposal';
newNote.Description = 'This is a comment';
newNote.NoteType = 'General';

this.noteService.addComment(newNote).subscribe(savedNote => {
});
Parameters :
Name Type Optional Description
note Note No
  • Note object containing Name, ObjectId, ObjectType, Description and optional NoteType
Returns : Observable<Note>

Observable of the created Note record.
getComments
getComments(recordId: string, recordType: string, page: number, pageSize: number, noteType?: string, sortField: string, sortDirection: "Ascending" | "Descending")

Fetches comments for a specific record with pagination, filtering, and sorting support

Example:

this.noteService.getComments('proposal123', 'Proposal').subscribe(comments => {
});

// With pagination and filtering
this.noteService.getComments('proposal123', 'Proposal', 1, 10, 'General')
    .subscribe(comments => this.commentsList = comments);
Parameters :
